# Flappy-Bird-next-gen
Built a replica of one of world's most favourite game. Helped me gain knowledge on game -development.
The design of the Flappy Bird game is based on a simple 2D game structure.
The game consists of a bird character, moving pipes as obstacles, a
background, and a scoring system. The bird continuously falls due to
gravity, and the player must press a key to make the bird flap upward. The
pipes move horizontally from right to left, and the player must guide the
bird through the gaps between the pipes.
The game uses a main game loop that continuously updates the screen,
checks for user input, and detects collisions. Collision detection is
implemented to check whether the bird hits a pipe or the ground, which
results in game over. A scoring system is implemented to increase the score
when the bird successfully passes through a set of pipes.
Pygame is used to handle graphics rendering, sound effects, and keyboard
events. The modular design of the program makes it easy to understand and
modify. The overall design ensures smooth gameplay and user interaction
while maintaining simplicity and efficiency.
